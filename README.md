# HGSL
Source code of AAAI submission "Heterogeneous Graph Structure Learning for Graph Neural Networks"
# Requirements
## Python Packages
- Python >= 3.6.8
- Pytorch >= 1.3.0
- DGL == 0.4.3
## GPU Memmory Requirements
- ACM >= 8G
- DBLP >=5G
- Yelp >=3G 
# Usage
Take DBLP dataset as an example:
python train.py --dataset='dblp'
